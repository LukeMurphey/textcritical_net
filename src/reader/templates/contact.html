Get In Touch

We want to here from you. Bug reports, feature requests, or just a thumbs up are all good.

Bug reports

We love bug reports (well, kind of). If you found something that doesn't work, please make a issue at LukeMurphey.net. You don't need an account to make a bug report so it should be a rather painless process. You will probably get a response to the issue within 24 hours. You might want to search the currently open issues to make sure it wasn't already reported.

Feature Requests

We are always on the lookout for useful new features to implement. Don't be shy, let us know what you think would make this site better. To do so, make a feature request and let us know what you would want to see. Your feature request will be evaluated and assigned to a release if accepted.
